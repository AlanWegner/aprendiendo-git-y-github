Cómo utilizar GIT

  • Primero - Nos posicionamos en la carpeta raíz.
  • Segundo - Ejecutamos el comando: 'git init', con el cual indicamos que estamos creando un repositorio en esta carpeta.
  • Tercero (opcional) - Ejecutamos 'git status -s', para ver el estado de nuestro repositorio de manera resumida.
  • Cuarto - Guardamos los archivos para que GIT esté pendende de los cambios, utilizándo el comando 'git add .'
  • Quinto - Etiquetamos todos los cambios que hemos agregado, con el comando 'git commit -m "Mensaje del commit"'
  • Sexto - Visualizamos todos los commit com el comando 'git log --oneline'
  • Séptimo - Realizamos la creación de un repositorio en la página principal de GitHub
  • Octavo - Realizamod el 'git push' para cargar todo al repo de internet, no el local
